# 새로운 분석 시작 페이지 재구성 수용 기준

## `AC-01` 셸 레이아웃

- 화면 진입 시 사이드바·상단 바·본문이 함께 보인다.
- 본문은 제목/서브카피와 좌(입력 단계)|우(안내 레일) 2단으로 배치된다.

## `AC-02` 사이드바·프로필·모의 면접

- 전체 메뉴와 모의 면접이 사이드바에 보인다.
- 홈·모의 면접만 활성이고, 잠금 항목은 이동이 되지 않는다.
- 계정/프로필·플랜 영역이 정적으로 표시된다(클릭해도 인증/결제 동작 없음).

## `AC-03` 3단계 입력

- ① JD 입력에서 `JD 링크`/`JD 내용 붙여넣기` 탭을 전환할 수 있고, 붙여넣기에는 글자 수가 표시된다.
- ② 이력서 업로드에서 파일을 올리면 파일 이름·크기가 보이고 제거할 수 있다(PDF·DOCX).
- ③ 분석 옵션을 체크박스로 선택/해제할 수 있다.

## `AC-04` 분석 옵션 동작 매핑

- 옵션은 `JD 적합도`·`ATS 분석`·`문장 첨삭`·`키워드 분석`로 표시된다.
- `JD 적합도`는 실제 매칭을 실행한다.
- `ATS 분석`·`문장 첨삭`·`키워드 분석`은 선택해도 "준비중"으로 안내되고 외부 호출이 없다.

## `AC-05` 분석 실행과 결과

- `분석 시작하기`를 누르면 JD·이력서·옵션 검증 후 분석이 실행된다.
- `JD 적합도` 결과(매칭 점수·요약·강점·gap·제안)가 결과 화면에 표시된다.
- 준비중 옵션을 선택했으면 결과 화면에 "준비중" 패널로 표시된다.

## `AC-06` 계약·검증·에러 보존

- 진단/매칭/면접/JD수집 API 요청·응답 구조가 변경되지 않는다.
- 빈/짧은/긴 입력 검증, 키 누락·Gemini 실패 에러 처리가 기존과 동일하게 동작한다.
- `fixture` 모드에서 외부 호출 없이 결과가 정상 표시된다.

## `AC-07` 반응형·안내 레일·접근성

- 우측 안내 레일(진행 방식·정확도 팁·지원 형식)이 표시된다.
- 데스크톱은 입력|안내 2단, 좁은 폭은 세로 스택으로 전환된다.
- 주요 입력·버튼·결과의 label/role/포커스/오류 알림이 유지된다.
