# 요구사항

> 프로젝트: JDSnack — AI 이력서 진단 서비스  
> 상태: 활성  
> 기준 문서: 기존 `shape.md`, `mission.md`, `roadmap.md`

## 사용자 문제

취업 준비생과 개발자는 이력서를 작성해도 가독성, 프로젝트 기여도, 성과 표현이 충분한지 빠르게 검토받기 어렵습니다.

## 기능 목표

사용자가 이력서 텍스트를 입력하면 AI가 이를 분석해 점수와 개선 피드백을 즉시 제공합니다.

## 요구사항

### `REQ-01` 이력서 텍스트 진단

- 사용자는 이력서 텍스트를 입력하고 진단을 요청할 수 있어야 한다.

### `REQ-02` 구조화된 결과 제공

- 시스템은 점수, 가독성 피드백, 프로젝트 기여도 피드백, 종합 요약을 구조화된 형태로 제공해야 한다.

### `REQ-03` 예외 상황 안내

- 빈 입력, 너무 짧은 입력, 너무 긴 입력, 외부 AI 실패에 대해 사용자 친화적 메시지를 제공해야 한다.

### `REQ-04` 단순한 운영 구조

- MVP는 단일 JAR 배포 구조를 기본값으로 하여 운영 복잡도를 낮춰야 한다.

## 범위 포함

- 텍스트 입력 기반 이력서 진단
- AI 점수 및 피드백 리포트
- 기본 예외 처리
- 단일 JAR 배포 구조

## 범위 제외

- 회원가입/로그인
- 서버 DB 저장
- 과거 진단 히스토리 조회
- PDF/Word 직접 업로드 파싱
- JD 매칭 및 모의 면접 질문 생성
