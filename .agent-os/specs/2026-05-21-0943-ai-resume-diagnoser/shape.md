# 1차 MVP 형태

## 핵심 기능

사용자가 이력서 텍스트를 입력하고 진단 요청을 누르면, 시스템은 입력값을 검증한 뒤 AI 분석 기능이 준비 중임을 안내한다.

## 1차 MVP에서 하지 않는 것

- 사용자 인증 정보 입력
- 서버 외부 AI 연동
- AI 점수 생성
- AI 피드백 생성
- 분석 결과 저장

## 사용 흐름

1. 사용자가 웹 페이지에 접속한다.
2. 이력서 텍스트를 입력한다.
3. 진단 요청 버튼을 누른다.
4. 프론트가 `POST /api/diagnose`를 호출한다.
5. 백엔드가 입력값을 검증한다.
6. 검증 실패 시 입력 기준을 안내한다.
7. 검증 성공 시 `AI_ANALYSIS_NOT_ENABLED` 준비중 메시지를 안내한다.

## 후속 확장

2차 MVP에서 서버 환경변수 기반 외부 AI 연동과 실제 분석 결과 생성을 추가한다.
