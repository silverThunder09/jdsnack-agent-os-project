# AI 이력서 진단 서비스 참고 아키텍처

본 프로젝트는 완전히 신규 구축하는 서비스로 직접 참조할 기존 코드는 없으나, 다음의 최신 웹 아키텍처 디자인 패턴과 모던 AI 연동 가이드라인을 근거로 개발을 진행합니다.

## 외부 참고 아키텍처 디자인 패턴

### 1. Apple 디자인 시스템 (Apple Human Interface Guidelines)
- **핵심 패턴**: 정돈된 둥근 모서리 카드, 넓은 흰색 공백, 얇고 가독성 높은 폰트 두께, 섀도우를 통한 레이어 분리.
- **적용**: 이력서 입력 영역과 진단 결과 분석 보고서 영역의 레이어 분리에 적용.

### 2. Client-side AI Integration Pattern
- **핵심 패턴**: 브라우저에서 직접 AI 연동 시 로딩 바 및 분석 진행도를 세밀하게 연출하여 대기 시간 동안의 사용자 이탈 방지.
- **적용**: 진단 실행 시 "AI 분석 중..." 스피너 및 실시간 점수 게이지 카운팅 효과 구현.
