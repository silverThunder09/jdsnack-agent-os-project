# 추적 매핑표

| 요구사항 | 수용 기준 | 테스트 시나리오 | 계약 / 설계 문서 |

## 사용 규칙

- 요구사항이 바뀌면 이 표를 먼저 갱신합니다.
- 테스트가 추가되면 대응하는 `REQ`와 `AC`를 연결합니다.
- 이후 실제 코드가 생기면 테스트 클래스/파일 경로 컬럼을 추가합니다.
