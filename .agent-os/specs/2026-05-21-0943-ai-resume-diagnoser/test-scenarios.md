# 테스트 시나리오

> 기능: AI Resume Diagnoser MVP

## `TC-01` 정상 이력서 진단

- 대응 AC: `AC-01`, `AC-06`
- 절차:
  - 50자 이상 이력서 텍스트 입력
  - 진단 버튼 클릭
- 기대 결과:
  - 200 응답
  - 점수/피드백/요약 표시

## `TC-02` 빈 입력 요청

- 대응 AC: `AC-02`
- 절차:
  - 입력 없이 요청
- 기대 결과:
  - 검증 실패
  - 사용자 안내 메시지 표시

## `TC-03` 너무 짧은 입력

- 대응 AC: `AC-03`
- 절차:
  - 50자 미만 텍스트 입력
- 기대 결과:
  - 검증 실패
  - 길이 기준 메시지 표시

## `TC-04` 너무 긴 입력

- 대응 AC: `AC-04`
- 절차:
  - 10,000자 초과 텍스트 입력
- 기대 결과:
  - 검증 실패
  - 최대 길이 기준 메시지 표시

## `TC-05` 외부 AI 실패

- 대응 AC: `AC-05`
- 절차:
  - Gemini API 실패 상황 모의
- 기대 결과:
  - 사용자 친화적 오류 메시지 반환
  - 내부 예외 세부사항 비노출

## `TC-06` 문서 계약 동기화 검토

- 대응 AC: `AC-07`
- 절차:
  - API/UI/요구사항 변경 시 관련 문서 갱신 여부 확인
- 기대 결과:
  - 누락 문서 없이 변경 세트가 완성됨
