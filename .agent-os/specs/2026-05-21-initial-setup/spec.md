# 초기 세팅 명세

## 목표

JDSnack 메인 저장소의 기본 구조를 만들고, 백엔드/프론트엔드/문서가 함께 움직일 수 있는 기반을 준비합니다.

## 범위

- 루트 저장소 구조 정리
- `.agent-os` 문서 체계 구성
- `backend/`, `frontend/`, `docs/`, `scripts/` 경로 확보
- 초기 안내 문서 작성

## 제외 범위

- 실제 Spring Boot 프로젝트 생성
- 실제 React 프로젝트 생성
- 배포 설정

## 산출물

- 디렉토리 구조
- 루트 안내 문서
- 제품/표준/에이전트 문서 초안
