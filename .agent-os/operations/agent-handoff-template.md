# Agent Handoff 템플릿

아래 형식으로 각 에이전트의 작업 종료 내용을 남깁니다.

```md
# Agent Handoff

## Feature

- 기능명:

## Current Phase

- Spec / Backend / Frontend / QA / Security / DevOps / Release

## Source Documents

- 참조한 문서:

## Changed Files

- 변경한 파일:

## Decisions Made

- 결정한 내용:

## Change Requests

- 다른 에이전트에게 요청할 변경 사항:

## Open Questions

- 남은 질문:

## Risks

- 남은 위험:

## Next Agent

- 다음 담당 에이전트:
```

## 작성 규칙

- 변경한 파일은 실제 경로로 적습니다.
- 변경 요청은 담당 에이전트 이름과 함께 적습니다.
- 위험은 머지 전 해결 필요 여부를 표시합니다.
- 질문이 없으면 `없음`으로 적습니다.
