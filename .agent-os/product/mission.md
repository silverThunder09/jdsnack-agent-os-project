# 제품 미션 (Product Mission)

## 문제 정의 (Problem)

많은 취업 준비생과 이직을 준비하는 개발자들이 자신의 이력서(Resume)를 작성할 때 가독성이 떨어지거나, 프로젝트 기여도 및 구체적인 성과가 드러나지 않아 서류 전형에서 탈락하는 어려움을 겪습니다. 전문가의 피드백을 받기에는 비용과 시간이 많이 소모됩니다.

## 타겟 사용자 (Target Users)

직무 기술서(JD)에 맞추어 자신의 강점을 극대화하고, 이력서의 가독성과 설득력을 높여 서류 합격률을 향상시키고자 하는 신입 및 경력 개발자들과 구직자들입니다.

## 해결 방법 (Solution)

JDSnack은 사용자가 자신의 이력서를 텍스트로 입력하면 단계적으로 이력서 검증, 분석, 개선 피드백을 제공하는 **AI 이력서 진단 및 개선 가이드 웹 서비스**입니다.

1차 MVP에서 입력 화면, 검증 API, 준비중 안내 흐름을 완성했고, 현재는 2차로 진입해 서버 기반 **Gemini 연동(`ai-local` 모드)으로 이력서 진단과 JD 적합도 매칭**을 실제로 제공합니다. ATS 분석·문장 첨삭·키워드 분석은 다음 단계로 순차 확장하며 현재는 준비중 안내로 표시합니다.
