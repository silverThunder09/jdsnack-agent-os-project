# 결과 내보내기·입력 자동저장 수용 기준

## `AC-01` 결과 내보내기

- 결과(성공) 상태에서 "내보내기"와 "인쇄" 동작이 보인다.
- 내보내기를 누르면 점수·요약·강점·gap·제안이 담긴 마크다운(.md) 파일이 내려받아진다.
- 인쇄를 누르면 브라우저 인쇄가 호출된다.
- 결과가 없는 상태(분석 전/실패)에서는 내보내기·인쇄가 노출되지 않거나 비활성이다.

## `AC-02` 입력 자동저장·복원

- JD 본문과 분석 옵션을 입력한 뒤 새로고침하면 그 값이 자동 복원된다.
- 이력서 파일은 복원되지 않는다(다시 업로드해야 한다).
- "입력 초기화"를 누르면 저장된 JD 본문·옵션이 비워지고 로컬 저장도 삭제된다.

## `AC-03` 계약·검증·접근성 보존

- 진단/매칭/면접/JD수집 API 요청·응답 구조가 변경되지 않는다.
- 빈/짧은 입력 검증, 키 누락·Gemini 실패 에러 처리, `fixture` 모드 동작이 기존과 동일하다.
- 추가된 버튼은 명확한 label/role을 가지며, 데스크톱·좁은 폭 레이아웃이 유지된다.
